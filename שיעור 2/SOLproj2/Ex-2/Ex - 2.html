Exercise #2

Stocks Application

 

Exercise targets

      1. Experience the Visual Studio .NET further on and on your own (learn how to tame this beast).

      2. Work with variables, operators and conditions.

What you should do

You are expected to write a console application with the following capabilities:

  • Print a message that asks the user to type his first name.
  • Receive the data from the user.
  • Display a message with 3 options as folows:
      Choose your option:
      1 - Buy a stock.
      2 - Buy stocks on sales.    .
  • According to the user's input do the following:
      If the user chose to buy stocks:
       1. Print a message asking the user to insert the price of the desired stock (whole numbers).
       2. Receive the data from the user.
       3. Print a message asking the user to insert the number of the desired stocks.
       4. Receive the data from the user.
       5. If the number of stocks is even, than calculate the total amount to pay and print to the screen.
       6. If the number of stocks is odd, than print a message informing that this application cannot calculate odd numbers of stocks.

      If the user chose to buy stocks on sale:
       1. Print a message asking the usr to insert the regular price of the desired stock (price for one stock).
       2. Receive the data from the user.
       3. Print a message asking the user to insert the amount of the desired stock he would like to purchase on sale.
       4. Receive the data from the user.
       5. If the number of stocks is more than 10    6. If the number of stocks is exactly 10, than each stock will cost 20% off the regular price (20% discount).
       7. If the number of stocks is less than 10, than there will be no discount..
       8. Print the data as follows:     Stock regular price is: |stock regular price|.
        Stocks amount: |stocks amount|.
        Stock regular price is: |stock regular price|.
        Discount percent is: |dicsount percent|.(This number will be printed as a whole precent: 0.2 will be 20%)
        Stock price after discount is: |stock price after discount|.
        Total amount to pay: |total cost of stocks purchased on sale|.

       If the user chose a number that is not listed in the options above:
       1. Print the following message to the user:
        This option is not a valid option!

Do not forget to write some documentations ( use // or /**/ )

Indentation is crucial (use the right tabing)

What you should not do

Whenever the user is prompt to insert a value, you can take from granted that the value inserted is valid. For example if the user is suppose to insert an integer and instead he inserts a string it is not up to you to handle the error (or at least not for this exercise).

Special guidelines

Please stick to the same indentation conventions we use in our examples and which the Visual Studio .NET help create. Use a clear and understandable names for your variables, for example the best name for the user's name will be (surprisingly) userName.
